# Changelog

All notable changes to **cp_waterfight** are documented here.

## 1.0.0

Initial playable release.

- Arena setup and validation (`create`, `setlobby`, `setjoin`, `addspawn`, `validate`, `info`, `list`)
- Join block and TextDisplay with live player count and state
- Countdown and full game flow (waiting, countdown, in-game, ending)
- 20-level weapon progression (GunGame-style kills per level)
- Kill tracking, level-up, and win flow
- Sidebar scoreboard with rank, level, weapon, and top players
- Protection system (inventory, blocks, commands, durability, pre-start damage)
- Spectator respawn polish after in-game deaths
- Sounds, titles, and actionbar feedback
- Debug and version commands
- Empty arena auto-reset when all players leave
- Death drop cleanup during active matches
- Protection message throttling
